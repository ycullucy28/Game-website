# Audio

Original chiptune-style audio for the game site. Files are generated by `scripts/generate_audio.py`.

## Sound Effect Style Options

The **Sound effect style** setting controls which sounds play:

| Style | Theme music | Move / score | Game over |
|-------|-------------|--------------|-----------|
| **Classic** | Yes (classic theme) | No | Yes |
| **Retro** | Yes (retro theme) | No | Yes |
| **Minimal** | No | Yes | Yes |

- **Mute background music** only affects theme music on Classic and Retro.
- **Enable sound effects** controls move, score, and game over sounds.

## Files

**Action sounds (Minimal mode):**

- `move.wav`
- `score.wav`
- `game-over.wav`

**Theme music (Classic / Retro modes):**

- `snake-theme-classic.wav`
- `snake-theme-retro.wav`
- `tetris-theme-classic.wav`
- `tetris-theme-retro.wav`

## Regenerate

```bash
python3 scripts/generate_audio.py
```
